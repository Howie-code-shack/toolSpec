# Example Custom Skill

This is an example skill file. Skills are reusable procedures or commands that Claude can execute in your project.

## What Are Skills?

Skills are specialized capabilities that you want Claude to perform consistently. They can be:
- Custom commands or scripts
- Multi-step procedures
- Project-specific workflows
- Automated tasks

## Skill Template

```markdown
# Skill Name: [descriptive-name]

## Description
Brief description of what this skill does and when to use it.

## Usage
/skill-name [arguments]

## Steps
1. First step of the procedure
2. Second step
3. Third step

## Examples
Example usage and expected outcomes

## Prerequisites
- Required tools or dependencies
- Environment setup needed

## Notes
- Important considerations
- Common pitfalls
- Related skills
```

## Example Skills You Might Create

- `/deploy-staging` - Deploy to staging environment
- `/run-full-test-suite` - Run all tests with coverage
- `/generate-api-docs` - Generate API documentation
- `/setup-dev-env` - Set up local development environment

Delete this example file and create your own skills as needed.
